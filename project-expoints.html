Project Overview

EXPoints is a comprehensive, PHP-based web platform that serves as a centralized hub for gamers to browse, search, and read trustworthy video game reviews across all platforms and eras. Registered users can submit their own reviews featuring detailed text and a star rating, as well as comment on other users' posts to foster community discussion.


What sets EXPoints apart is its unique gamified experience layer. Users earn XP Points for meaningful interactions, including posting reviews, commenting, and receiving "StarUps" (upvotes) from peers. These points contribute to an overarching user level, which is showcased on personalized profile pages alongside their activity history.


To ensure a safe and respectful community, EXPoints implements a strict Role-Based Access Authentication system. The system defines clear, tiered privileges for Admins, Moderators, Registered Users, and Guests. While user data is stored securely in Supabase, Admins and Moderators utilize specialized dashboard tools to manage accounts, review user-reported content, and remove inappropriate posts.
